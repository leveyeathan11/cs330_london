## How do I approach designing software? How do I approach developing programs? How can computer science help me in reaching my goals?

In any computer project, having a holistic approach and working without bias is essential to the success of the final output. I started this project by brainstorming many of the details before even beginning, and tried to envision 
from the start what simplifications would need to be made to accomplish the scene. Despite much planning, there were still road blocks, but I overcame them through perseverence and consulting documentation on the libraries being used.
In addition to the creative aspects of this project, the principles of object-oriented programming were quite valuable to keeping my code readable and modularized. I broke down each object in my scene into a separate function for the sake of organization,
which also allowed me to reuse objects without repeating code. I was familiar with this aspect of coding before taking this course, but I was given the opportunity to learn about topics such as lighting in this course which were new to me. I plan on pursuing game development,
so this experience with lighting and collisions will be extremely valuable to me as I continue to learn new skills and grow. I plan to first start by expanding my knowledge of other graphics libraries beyond OpenGL.
